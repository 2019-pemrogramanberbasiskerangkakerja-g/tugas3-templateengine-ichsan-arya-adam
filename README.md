### Cara Menjalankan
1. Clone Repositori ini.
2. Pindah ke direktori ExpressJs-Handlebars-Example
3. run ``` npm install ``` untuk mengunduh semua dependencies yang dibutuhkan
4. run ``` node app.js ```
5. Buka browser anda jalankan http://localhost:3000/
